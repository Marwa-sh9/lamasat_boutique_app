# 👗 تطبيق لمسات بوتيك - Lamasat Boutique App

تطبيق موبايل متطور مبني باستخدام **Flutter** مخصص لعرض وبيع وتأجير الفساتين والأزياء، مع نظام متكامل لإدارة مقاسات الزبائن وتجربة مستخدم سلسة.

---

## ✨ المميزات الرئيسية | Key Features
* **🛍️ نظام البيع والتأجير:** تصفح واختيار الفساتين مع تفاصيل كاملة.
* **📏 إدارة المقاسات:** ملف خاص لكل زبونة يحتوي على مقاساتها الدقيقة.
* **🔐 الحماية والخصوصية:** تسجيل دخول آمن عبر Google و Firebase.
* **☁️ إدارة الصور:** رفع ومعالجة الصور بكفاءة عبر Cloudinary.
* **🛡️ لوحة تحكم الأدمن:** إدارة شاملة للمنتجات، الأقسام، الطلبات، والمستخدمين.

---

## 🛠 الجانب التقني | Technical Stack
* **State Management:** BLoC / Cubit.
* **Backend:** Firebase (Firestore, Auth, Realtime Database).
* **Validation:** نظام تحقق ذكي من البيانات مع Loading States احترافية.
