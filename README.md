# 👗 تطبيق لمسات بوتيك - Lamasat Boutique App

تطبيق موبايل متطور مبني باستخدام **Flutter** مخصص لعرض وبيع وتأجير الفساتين والأزياء، مع نظام متكامل لإدارة مقاسات الزبائن.

---

## ✨ المميزات الرئيسية | Key Features
* **🛍️ نظام البيع والتأجير:** تصفح واختيار الفساتين بسهولة.
* **📏 إدارة المقاسات:** ملف خاص لكل زبونة يحتوي على مقاساتها الدقيقة.
* **🔐 الحماية:** تسجيل دخول عبر Google و Firebase.
* **☁️ الأداء:** رفع الصور ومعالجتها عبر Cloudinary.
* **🛡️ لوحة التحكم:** إدارة كاملة للأقسام والطلبات من قبل الأدمن.

---

## 🛠 الجانب التقني | Technical Stack
* **State Management:** BLoC / Cubit.
* **Backend:** Firebase (Firestore, Auth, Database).
* **Validation:** نظام تحقق ذكي من المدخلات مع Loading States احترافية.
